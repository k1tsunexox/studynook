## ✿ About

StudyNook is a full-stack academic productivity platform designed to bring everything students need into one organized workspace.

Instead of switching between multiple apps for schedules, notes, flashcards, assignments, calendars, and study sessions, StudyNook keeps your entire academic workflow together in one modern application.

Built with **Next.js App Router**, **Server Actions**, **Supabase**, **Drizzle ORM**, and **TypeScript**, it focuses on delivering a fast, clean, and enjoyable study experience.

## ✿ Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/studynook.git

cd studynook

npm install

npm run dev
```

Open **http://localhost:3000** in your browser.
